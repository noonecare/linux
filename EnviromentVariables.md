# Important Enviroment Variable

# PATH： 可执行文件的搜索目录
# LANG

```shell
locale
```
locale 显示本地化信息，是 12 个与本地化相关的环境变量的值

-----

# C 语言

LD_LIBRARY_PATH: lib 文件的搜索目录，找不到 lib 文件时常常是这个环境变量设的不对

----

# JAVA 语言

CLASS_PATH: 类搜索路径

----

PYTHON 语言

PYTHONPATH: PYTHON module 搜索路径


很多软件的 HOME 路径，比如 SPARK_HOME, JAVA_HOME, HADOOP_HOME 等路径
